Wireshark GUI Wrapper

A friendly, easy window for looking at your network traffic on Linux. No typing commands, no technical know-how required - just click and go.

Download the App Setup Helper Instructions

Save all three into the same folder on your computer.

Getting started

  1. Save the three files above into one folder.
  2. Open a terminal in that folder and type: bash install.sh — this checks your computer has everything it needs, and asks before installing anything.
  3. Then type: python3 wireshark_gui.py to open the app.
  4. A window will appear - you're ready to go!

Using the app

🔌
Interface This is just which internet connection to watch (Wi-Fi, wired, etc). Pick one from the dropdown at the top.
🔍
Filter (optional) Want to see only certain traffic? Type something simple like http to see web traffic. Leave it blank to see everything.
▶️
Start / Stop Click Start to begin watching traffic - it will show up in a list as it happens. Click Stop whenever you're done.
🧹
Clear Wipes the list on your screen so you can start fresh. It doesn't stop or delete your capture.
💾
Save As... Saves what you captured as a file you can keep, send, or open later.
🔬
Open in Wireshark If you need to look closer, this opens your capture in the full Wireshark program for more detail.
💡 Good to know Watching network traffic usually needs extra permission on Linux. If the app can't start capturing, just run it with sudo python3 wireshark_gui.py instead, or follow the one-time setup notes in the Instructions file so you don't need to do that every time.

Runs entirely on your own computer - nothing is ever uploaded. Only use this on networks and devices you're allowed to monitor.
