Wireshark GUI Wrapper

A simple point-and-click app for capturing network traffic on Linux - no command line needed.

App (.py) Setup script README

Quick start

  1. Download all three files above into the same folder on your Linux computer.
  2. Open a terminal in that folder and run bash install.sh to check/install what's needed.
  3. Launch the app with python3 wireshark_gui.py.
  4. Pick a network interface, click Start, and watch traffic appear.

Runs entirely on your own computer. Only capture traffic you own or are authorized to monitor.
