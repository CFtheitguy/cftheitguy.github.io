Wireshark GUI Wrapper

A point-and-click packet capture tool for Linux. It runs tshark (Wireshark's capture engine) behind a simple window with Start/Stop, a filter box, and a live packet table - no terminal commands to memorize.

Download

wireshark_gui.py install.sh README.md

Save all three files into the same folder on your Linux machine.

Setup

  1. Open a terminal in the folder where you saved the files.
  2. Run the guided setup script - it checks for missing packages and asks before installing anything: 
    bash install.sh
  3. Launch the app: 
    python3 wireshark_gui.py

How to use it

  • Interface — the dropdown at the top lists your network interfaces (e.g. eth0, wlan0). Pick the one you want to monitor, or hit Refresh if it just changed.
  • Filter — type a normal Wireshark display filter here to narrow what you see, e.g. tcp.port == 443, http, or ip.addr == 192.168.1.10. Leave it blank to capture everything.
  • Start / Stop — begins or ends the capture. Packets appear live in the table below as they're seen: number, time, source, destination, protocol, length, and info.
  • Clear — empties the table on screen (doesn't affect a running capture).
  • Save As... — exports the capture to a .pcapng file you can keep or share.
  • Open in Wireshark — launches the full Wireshark app on the same capture, for deeper inspection than the simple table offers.

Capturing without sudo every time

Packet capture normally needs root. To let your regular user account capture instead of running the app as root, run once:

sudo dpkg-reconfigure wireshark-common   # choose "Yes"
sudo usermod -aG wireshark $USER

Then log out and back in. Otherwise, just run sudo python3 wireshark_gui.py.

Only capture traffic on networks and devices you own or have explicit permission to monitor.

Runs entirely on your own computer. Nothing is uploaded anywhere.
